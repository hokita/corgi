Auth Middleware — Request flow

REJECT 401No Authorization header or token is empty
REJECT 401getAuth().verifyIdToken(token) throws (expired / tampered / wrong project)
REJECT 401decoded.email is undefined (phone auth / anonymous user)
REJECT 401decoded.email !== process.env.ALLOWED_EMAIL
PASS ✓Set req.uid = decoded.uid → all Firestore queries scoped to this uid
