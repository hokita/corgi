Auth Flow

① OPENApp.tsx calls onAuthStateChanged(auth, setUser)
② LOADINGuser === undefined → return null (prevents screen flash)
③ LOGGED OUTuser === null → redirect to /login
④ SIGN INTap button → signInWithPopup(auth, new GoogleAuthProvider())
⑤ LOGGED INFirebase stores session · onAuthStateChanged fires → redirect to /
⑥ API CALLSauth.currentUser.getIdToken() → fresh JWT auto-attached to every request
